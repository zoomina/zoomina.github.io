    ### From GA/GTM & Meta Pixel Setup to YouTube API Market Analysis and UI Optimization
    #Python #EDA #FunnelAnalysis #DataVisualization #Dashboarding #GoogleAnalytics #GTM #PMFValidation
    <br>
    <strong>Problem</strong>
    <ul style="list-style:none; padding-left:0;">
      <li>This project addressed low subscription conversion after paid traffic acquisition.</li>
      <li>The goal was to identify where users dropped off on the landing page and improve the signup funnel through data-driven UX changes.</li>
    </ul>
    <br>
    <strong>Data</strong>
    <ul>
      <li>Google Analytics and Google Tag Manager event data</li>
      <li>Meta Pixel-based paid traffic data</li>
      <li>YouTube API data for liquor-related content demand analysis</li>
    </ul>
    <br>
    <strong>Method</strong>
    <ul>
      <li>Funnel analysis across landing, CTA exposure, and subscription events</li>
      <li>Event-based behavioral analysis to detect low-visibility and drop-off points</li>
      <li>Additional content demand analysis using external platform data</li>
    </ul>
    <br>
    <strong>Insights</strong>
    <ul>
      <li>The subscription form had low actual visibility because it was placed below the fold</li>
      <li>Repositioning the CTA above the fold improved conversion efficiency significantly</li>
      <li>Subscription conversion increased by 170%, while CPA dropped by about 60%</li>
    </ul>
