## 문자열 표시하기
----

#### 1. 문자열을 표시하는 방법

Python은 C나 Java와 달리 문자열을 표시할 때 `'`와 `"`를 구분하지 않습니다.  

```python
str1 = 'Hello'
str2 = "Python"

print(str1, str2)

>>> 출력결과
Hello Python
```

또한 `'''` 혹은 `"""`을 사용하면 raw string을 사용할 수 있습니다.  
키보드에서 입력한 그대로 보여주기 때문에 개행문자`\n`를 사용할 필요가 없습니다.  

```python
str1 = 'Hello\nI\nam\nMINA'
str2 = '''Nice
to
meet
you'''
print(str1)
print(str2)

>>> 출력결과
Hello
I
am
MINA
Nice
to
meet
you
```

<br>

#### 2. 이스케이프 문자

이스케이프 문자는 앞서 사용했던 개행문자`\n`와 같이 특수한 기능을 하는 문자입니다.  
공통점은 모두 `\`를 사용한다는 것입니다.  
(참고로 `\`는 키보드에서 \\입니다.)  
종류는 매우 다양하지만 자주 사용하게 되는 몇 가지만 소개하겠습니다.  

문자 | 기능
:---: | :---:
`\n` | 개행문자(엔터)
`\t` | 탭(tab)
`\\` | 역슬래쉬 입력(\\)
`\'` | 작은따옴표 입력(')
`\"` | 큰따옴표 입력(")

```python
str = 'I said \t \"Hello!\"\nYou \tSmiled'
print(str)

>>> 출력결과
I said 	 "Hello!"
You 	Smiled
```

<br>

#### 3. 문자열 연산

Python에서는 `+`, `*` 연산을 이용해 문자열을 합할 수 있습니다.  
```python
str1 = "Let\'s "
str2 = "study "
str3 = "Python "
str4 = "!" * 3
str5 = str1 + str2 + str3 + str4
print(str5)

>>> 출력결과
Let`s study Python !!!
```

<br>

## 문자열 슬라이싱
----

이번에는 문자열을 슬라이싱 하는 두 가지 방법을 살펴보겠습니다.  
그 이전에 간단하게 문자열에 대해 살펴보도록 하겠습니다.  

문자열은 각자 0부터 시작하는 인덱스값을 가지고 있습니다.  

```
Hello MINA
0123456789
```

이를 활용하면 문자 하나하나에 쉽게 접근할 수 있습니다.

```python
str = "Hello MINA"
print(str[0])
print(str[3])
print(str[9])

>>> 출력결과
H
l
A
```

#### 1. 인덱스를 이용해 문자열 자르기

첫 번째로 살펴볼 방법은 위에서 설명한 인덱스를 이용한 방법입니다.  
위에서는 인덱스를 이용해 문자를 하나씩 불러왔는데요, 이번에는 여러 단어를 잘라내는 방법을 살펴보도록 하겠습니다.  

`[1:4]` : 1번 인덱스부터 4번 인덱스 전까지 잘라냅니다.  
`[:3]` : 처음부터 3번 인덱스 전까지 잘라냅니다.  
`[6:]` : 6번 인덱스부터 끝까지 잘라냅니다.  

**뒷쪽 인덱스는 포함하지 않습니다.**  

아래의 예제 코드를 통해 확인해보도록 하겠습니다.  

```python
str = "Hello MINA"
print(str[1:4])
print(str[:3])
print(str[6:])

>>> 출력결과
ell
Hel
MINA
```

<br>

#### 2. split() 함수를 이용해 문자열 자르기

인덱스가 아니라 특정한 문자를 기준으로 문자열을 나누어야 할 필요가 있는 경우가 있습니다.  
이 경우 사용하는 것이 split() 함수입니다.  
split() 함수를 사용할 경우 기본적으로 띄어쓰기를 기준으로 문자열을 잘라서 리스트에 정리해줍니다.  
띄어쓰기가 아닌 특정 문자를 기준으로 나누기를 원한다면 파라미터로 넘겨주면 됩니다.  

```python
str = "빨강, 주황, 노랑, 초록, 파랑, 남색, 보라"
slist1 = str.split()
slist2 = str.split(',')
print(slist1)
print(slist2)

>>> 출력결과
['빨강,', '주황,', '노랑,', '초록,', '파랑,', '남색,', '보라']
['빨강', ' 주황', ' 노랑', ' 초록', ' 파랑', ' 남색', ' 보라']
```

두 결과물의 차이를 보자면 slist1에서는 `,`가 지워지지 않았고, slist2에서는 ` `가 지워지지 않았습니다.  

그렇다면 깔끔하게 두 가지 이상의 문자를 기준으로 지워낼 수는 없을까요?  
이런 경우 re라는 모듈에 내장된 split() 함수를 사용하면 됩니다.  

```python
import re

str = "빨강, 주황, 노랑; 초록, 파랑, 남색- 보라"
slist = re.split(', |; |- ', str)
print(slist)


>>> 출력결과
['빨강', '주황', '노랑', '초록', '파랑', '남색', '보라']
```

이 함수를 사용하는 방법입니다.  
1. re 모듈을 import합니다.  
2. re.split('구분자(|를 이용해 구분)', 문자) 를 통해 구분자와 문자를 입력합니다.
