**예제 4-1**  
주민등록번호 뒷자리 첫 번째까지 입력받고 생일과 성별을 출력하는 프로그램을 작성하세요.  
조건1. 주민등록번호가 바르게 입력되었는지 확인해주세요.  
(길이와 형식`-`을 확인해주세요.)  
조건2. 주민등록번호 뒷자리 첫 번째 숫자를 확인해 1900년대인지 2000년대인지 확인해서 출력해주세요.  

**출력 예시**  
```
주민등록번호 뒷자리 첫 번째 숫자까지 입력해주세요. (oooooo-o):

(error) 형식에 맞추어 입력해주세요.
(error) 주민등록번호 뒷자리 첫 번째까지 입력해주세요.

생일: oooo.oo.oo.
성별: 남/여
```

<br>

**예제 답안**  

```python
# idNum이라는 변수에 주민등록번호 뒷자리 첫 번째 숫자까지 입력받습니다.
idNum = input('주민등록번호 뒷자리 첫 번째 숫자까지 입력해주세요. (oooooo-o):')

# len() 길이를 확인하는 함수
if len(idNum) != 8:
    print('(error) 주민등록번호 뒷자리 첫 번째까지 입력해주세요.')
# idNum의 idx 6번째 글자가 -가 맞는지 확인
elif idNum[6] != '-':
    print('(error) 형식에 맞추어 입력해주세요.')
# 뒷자리 첫 번째 숫자가 1, 2, 3, 4 중의 하나가 맞는지 확인
elif idNum[7] != '1' and idNum[7] != '2' and idNum[7] != '3' and idNum[7] != '4':
    print('(error) 형식에 맞추어 입력해주세요.')

# 뒷자리 첫 번째 숫자를 보고 1900년대 생인지 2000년대 생인지 판별
if idNum[7] == '1' or idNum[7] == '2':
    birth = 1900
elif idNum[7] == '3' or idNum[7] == '4':
    birth = 2000
else:
    birth = 0

# 생년에 주민등록번호 앞자리 두 자리를 더한다.
# 이때, idNum은 String이기 때문에 int형으로 타입 캐스팅 해준다.
birth = birth + int(idNum[:2])

# 뒷자리 첫 번째 숫자를 보고 성별 구분
if idNum[7] == '1' or idNum[7] == '3':
    gender = '남'
elif idNum[7] == '2' or idNum[7] == '4':
    gender = '여'
else:
    gender = '?'

# 결과 출력
print('생일:', birth, '.', idNum[2:4], '.', idNum[4:6], '.')
print('성별: ', gender)
```

**예제 4-2**  
randint를 이용해 가위바위보를 하는 프로그램을 작성하세요.  
조건1. 주먹:0, 가위:1, 보:2 로 입력받습니다.  
조건2. 출력할 때는 숫자로 출력합니다.  
randint의 사용법은 출력 예시 밑에 작성되어 있습니다.  

**출력 예시**  
```
주먹(0), 가위(1), 보(2)
가위 바위 보! :

(error) 정해진 숫자를 입력해주세요.

컴퓨터:
사용자:
______가 이겼습니다!
```

**randint 사용법**  
randint는 random이라는 모듈에 있는 int형을 랜덤으로 반환하는 함수입니다.  
```python
import random as r

# 0이상 10이하의 숫자를 랜덤으로 반환
num = r.randint(0, 10)
```

<br>

**예제 답안**  

```python
# import는 맨 첫 줄에 하는 것이 암묵적인 규칙입니다.
import random as r

# user의 결과는 입력과 동시에 int형으로 캐스팅합니다.
print('주먹(0), 가위(1), 보(2)')
user = int(input('가위 바위 보! :'))

# 정확한 숫자가 입력되었는지 확인하고, 아닐 경우 다시 입력받습니다.
if user != 0 and user != 1 and user != 2:
    print('(error) 정해진 숫자를 입력해주세요.')
    print('주먹(0), 가위(1), 보(2)')
    user = int(input('가위 바위 보! :'))

# 컴퓨터에 0~2 사이의 정수를 입력합니다.
com = r.randint(0, 2)

# 결과 출력
print('컴퓨터:', com)
print('사용자:', user)

if user == com:
    print('비겼습니다.')
elif (user % 3) + 1 == com:
    print('사용자가 이겼습니다.')
else:
    print('컴퓨터가 이겼습니다.')
```
