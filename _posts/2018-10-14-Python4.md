## if - elif - else
----

Python에서도 기존의 언어들과 비슷하게 조건문에 if문을 사용합니다.  
else if 대신 elif를 사용한다는 점은 약간 다르기에 주의가 필요합니다.  

<br>

조건문은 여러가지 선택지를 제시해야하는 경우 사용되는 문법입니다.  
즉, 프로그래밍을 할 때 특정 조건에 따라 다른 결과를 도출해야 하는 경우에 사용하는 구문입니다.  

if문의 기본적인 구조는 다음과 같습니다.

```python
if <조건문>:
    <실행 코드>
elif <조건문>:
    <실행 코드>
else:
    <실행 코드>
```

기존에 C나 Java 등의 언어를 사용하시던 분들께는 생소한 모양일 수 있는데요, Python에서는 기본적으로 구문을 괄호`{ }`로 구분하지 않습니다.  
`tab` 혹은 띄어쓰기 4번과 같은 **들여쓰기**를 통해 구문을 구분하기 때문에 들여쓰기에 항상 주의해야합니다.  
<br>
그럼 다시 한 번 위의 구조를 살펴보도록 하겠습니다.  
코드는 가장 첫 번째 if부터 마지막 else까지 차례로 실행이 됩니다.  
단, if를 제외한 모든 구문은 생략할 수 있고, elif는 필요한 경우에만 추가적으로 사용합니다.  

```python
a = 3

if a < 10:
    print('a < 10')

print('end')
```
```
>>> 출력 결과
a > 10
end
```

각 구문 블록에서 <조건문>은 True 혹은 False를 반환합니다.  
이때 Bool이 아닌 다른 타입으로 False를 표현하기 위해 `0` 혹은 비어있는 형태를 반환할 수 있습니다.  

조건문에서 True가 반환되면 아래의 실행 코드가 실행되고 뒤의 나머지 조건들은 실행하지 않습니다.  
그리고 False가 반환되면 다음 구문의 조건을 검토합니다.  
모든 조건들이 충족되지 않으면 else문을 실행하게 됩니다.  
만약 else가 없다면 아무 작업도 실행하지 않고 구문은 끝나게 됩니다.  

```python
a = 18

if a < 10:
    print('a > 10')

print('end')
```
```
>>> 출력 결과
end
```

## 연산자
----

그렇다면 if문의 조건을 위한 연산자를 살펴봐야겠죠.  
여기서는 두 가지 연산자를 살펴볼 예정입니다.  
조건문을 만들기 위한 **비교 연산자**와 여러개의 조건을 제시하기 위한 **논리 연산자**를 살펴보도록 하겠습니다.  

<br>

#### 1. 비교 연산자
비교 연산자는 다른 언어나 일상에서 사용하는 것과 비슷하게 사용하기에 크게 어렵지 않은 부분입니다.  

비교 연산자 | 설명
:---: | :---:
x < y | x가 y보다 작다
x <= y | x가 y보다 작거나 같다
x > y | x가 y보다 크다
x >= y | x가 y보다 크거나 같다
x == y | x와 y가 같다
x != y | x와 y가 같지 않다

프로그래밍을 처음하시는 분의 경우 등호`=`의 경우 이미 대입 연산자에서 사용하고 있기 때문에 비교 연산에서는 `==`으로 사용한다는 점에 주의해야합니다.  

<br>

#### 2. 논리 연산자

논리 연산자는 조건문 내에서 여러 개의 조건을 비교해주는 연산자 입니다.  
기존의 언어들과 달리 `and`, `or`, `not`과 같은 익숙한 예약어를 사용하고 의미도 우리가 알고있는 단어의 의미와 다르지 않기 때문에 크게 어렵지 않습니다.  

논리 연산자 | 결과
:---: | :---:
A and B | A와 B가 모두 True일 경우 True를 반환
A or B | A와 B 중 하나가 True일 경우 True를 반환
not A | A와 반대되는 결과를 반환

#### 예제 4-2
randint를 이용해 가위바위보를 하는 프로그램을 작성하세요.  
조건1. 주먹:0, 가위:1, 보:2로 입력받습니다.  
조건2. 출력할 때는 숫자가 아닌 글자로 출력합니다.  
randint의 사용법은 출력 예시 밑에 작성되어 있습니다.  

**출력 예시**
```
주먹(0), 가위(1), 보(2)
가위 바위 보! :

(error) 정해진 숫자를 입력해주세요.

컴퓨터:
사용자:
______가 이겼습니다!
```

**randint 사용법**  
randint는 random이라는 모듈에 있는 int형을 랜덤으로 반환하는 함수입니다.  

```python
import random as r

# 0이상 3이하의 숫자를 랜덤으로 반환
num = r.randint(0,10)
```
