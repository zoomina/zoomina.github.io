#### 예제 5-1
for문과 range()객체를 이용해 구구단을 작성하세요.  
<br>
**출력 예시**
```
2 * 1 = 1
2 * 2 = 2
...
9 * 8 = 72
9 * 9 = 81
```
<br>
**예제 답안**

```python
for dan in range(2, 10):
  for i in range(1, 10):
    print(dan, '*', i, '=', dan*i)
  print('')
```

#### 예제 5-2
nlist = [5, 2, 14, 3, 15, 26, 3, 20, 6, 18]  
nlist에서 최소값과 최대값을 출력하는 프로그램을 작성하세요.  
<br>
**출력 예시**
```
nlist = [5, 2, 14, 3, 15, 26, 3, 20, 6, 18]
최대값: 26
최소값: 2
```
<br>
**예제 답안**
```python
import math

# math.inf : 무한인 수를 표현
max = -math.inf
min = math.inf

nlist = [5, 2, 14, 3, 15, 26, 3, 20, 6, 18]

for n in nlist:
# min보다 작으면 min에 대입
    if n < min:
        min = n
# max보다 크면 max에 대입
    if n > max:
        max = n

print('최대값:', max)
print('최소값:', min)
```

#### 예제 5-3
지난 시간 풀었던 가위바위보 문제를 참고해서 사용자가 이길 때까지 반복하는 가위바위보를 작성하세요.  
조건1: 사용자가 이기면 게임이 종료됩니다.  
조건2: 잘못된 입력이 들어오면 다시 입력받으세요.  
<br>
**출력 예시**
```
주먹(0), 가위(1), 보(2)
가위바위보!:

(error) 잘못된 입력입니다.
주먹(0), 가위(1), 보(2)
가위바위보!:

컴퓨터:
사용자:
컴퓨터가 이겼습니다.
***
주먹(0), 가위(1), 보(2)
가위바위보!:

컴퓨터:
사용자:
사용자가 이겼습니다.
게임을 종료합니다.
```
<br>
**예제 답안**
```python
# import는 맨 첫 줄에 하는 것이 암묵적인 규칙입니다.
import random as r

while(True):
    # user의 결과는 입력과 동시에 int형으로 캐스팅합니다.
    print('주먹(0), 가위(1), 보(2)')
    user = int(input('가위 바위 보! :'))

    # 정확한 숫자가 입력되었는지 확인하고, 아닐 경우 다시 입력받습니다.
    if user != 0 and user != 1 and user != 2:
        print('(error) 정해진 숫자를 입력해주세요.')
        continue # 잘못된 입력 발생하면 다시 실행

    # 컴퓨터에 0~2 사이의 정수를 입력합니다.
    com = r.randint(0, 2)

    # 결과 출력
    print('컴퓨터:', com)
    print('사용자:', user)

    if user == com:
        print('비겼습니다.')
    elif (user % 3) + 1 == com:
        print('사용자가 이겼습니다.')
        break   # 사용자가 이기면 탈출
    else:
        print('컴퓨터가 이겼습니다.')
```
