## **커멘드 라인에서 작업하기**
----

IDLE를 실행하여 커멘드 라인을 이용해 대화하듯 작업할 수 있습니다.
- 간단한 계산을 진행할 수 있습니다.
```python
>>> 1 + 2
3
```

- 데이터 출력하기(print)
```python
>>> print("Hello World")
Hello World
```

, 를 이용해 여러 타입을 출력할 수 있습니다.
```python
>>> print("2 + 3 =", 2+3)
2 + 3 = 5
```

----
#### 에디터에서 작업을 시작하기 전에 살펴봅시다.

>1. 변수
>  - 자료의 저장소
>  - c나 java 등의 언어들과 달리 타입을 지정해주지 않아도 됨
```
>>>> num1 = 10
>>>> num2 = 20
>>>> num1 + num2
30
```

>2. 주석(#)
>  - 과제로 레포트를 제출할 때 주석은 내용에 영향을 주지 않지만 점수에는 영향을 줄 수 있다.
>  - 코드를 작성할 때, 주석은 **코드에 영향을 주지 않지만** 보다 읽기 편한 코드를 만들어준다.

>3. 데이터 타입
>   : python은 c나 java와 같은 기존의 언어들보다 간편한 데이터 타입을 제공한다.
>  - 정수형 : int
>  - 실수형 : float
>  - 문자열 : str
>  - 리스트 '[]' : list
>  - 튜플 '()' : tuple
>  - 딕셔너리 '{}' : dict
>
>  - 문자열과 리스트형 자료형에 대한 내용은 다음 포스트에서 자세히 다룰 예정입니다.  
>
>**- 데이터 타입 확인하기**
>
>```python
># type() 함수를 이용해 데이터 타입을 확인합니다.
>num1 = 20             # 정수
>num2 = 2.1            # 실수
>str = "Hello!"        # 문자열
>list = [1, 2, 3]      # 리스트
>tup = (1, 2, 3)       # 튜플
>dict = {1:'a', 2:'b'} # 딕셔너리
>print(type(num1))
>print(type(num2))
>print(type(str))
>print(type(list))
>print(type(tup))
>print(type(dict))
>```
>
>출력 결과
>```
><class 'int'>
><class 'float'>
><class 'str'>
><class 'list'>
><class 'tuple'>
><class 'dict'>
>```

## **에디터에서 작업하기**
----

IDLE의 [File]-[New File] 혹은 [Ctrl]+N 키를 이용해 에디터 창을 띄워 작업할 수 있습니다.

![image](https://user-images.githubusercontent.com/39390943/45596344-96077f80-b9f5-11e8-8232-95f51b6c69b2.png)

- 데이터 입력하기
```python
# input으로 입력받습니다.
name = input("이름을 알려주세요!")
print(name, "라고 하는구나! 이름이 정말 예뻐요!")
```
출력 결과
```
=================== RESTART: 경로 ===================
이름을 알려주세요!mina
mina 라고 하는구나! 이름이 정말 예뻐요!
```

- 모듈 불러오기
>import turtle as t  
> : 불러온다 turtle이라는 기능을, 그리고 t라는 약칭을 사용한다.

- 윈도우에서 그림판이라는 **프로그램**을 실행 = python에서 turtle이라는 **모듈**을 실행
- 그림판 **창** 안에서 작업 = 모듈의 **이름(혹은 약칭)**을 붙여 작업
- **버튼**을 클릭해 기능을 실행 = **함수/메서드**를 이용해 실행

```python
#t라는 약칭을 사용해 turtle을 실행할래요
import trutle as t

# 지름이 50인 원을 그려주세요.
t.circle(50)
```
