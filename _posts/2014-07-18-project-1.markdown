    ### MapleStory Meta Analytics Dashboard powered by Nexon Open API
    #Python #PostgreSQL #Airflow #Streamlit #Plotly #DataPipeline #GameAnalytics
    <br><br>
    <strong>Problem</strong>
    <ul style="list-style:none; padding-left:0;">
      <li>This project quantifies how game updates shift class balance and player fatigue in MapleStory.</li>
      <li>The goal was to move beyond community opinion and build objective indicators for meta change.</li>
    </ul>
    <br>
    <strong>Data</strong>
    <ul>
      <li>Nexon Open API-based ranking and character data</li>
      <li>Weekly Mu Lung Dojo records, equipment, hexacore, hyper stat, and ability data</li>
      <li>RAW-DW-DM layered pipeline for reproducible collection and analysis</li>
    </ul>
    <br>
    <strong>Method</strong>
    <ul>
      <li>Built a pipeline with Airflow, PostgreSQL, and staged data modeling</li>
      <li>Designed Balance Score using Shannon Entropy and Shift Score for patch impact</li>
      <li>Segmented users into 50F and high-tier groups to compare accessibility vs top-end meta</li>
    </ul>
    <br>
    <strong>Insights</strong>
    <ul>
      <li>Meta concentration and class dominance can be tracked quantitatively over time</li>
      <li>Patch impact is interpretable through outcome, investment, and build-change dimensions</li>
      <li>The framework supports balancing decisions with player-behavior-based evidence</li>
    </ul>
