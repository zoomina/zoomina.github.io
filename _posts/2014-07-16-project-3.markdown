    ### Shownote Brand Analysis using Twitter + KOPIS Ticketing Data
    #Python #NLP #WebCrawling #KOPISOpenAPI #DataPreprocessing #Tableau #DataVisualization
    <br>
    This project combines quantitative and qualitative research to assess Shownote’s brand positioning and propose actionable marketing directions. I collected Twitter data via crawling and applied Korean NLP (stemming + keyword filtering) to identify audience demand signals, visualized as a word cloud.
    <br>
    In parallel, I used the KOPIS Open API to analyze ticket sales and seat occupancy trends by show/date, and interpreted potential drivers such as cast exposure, media appearances, and seasonal effects (e.g., holidays). All preprocessing was done in Python, and the final dashboard was built in Tableau using time-series line charts and demand keyword visuals.

    쇼노트의 브랜드 포지셔닝을 점검하고, 개선이 필요한 지점과 마케팅 캠페인 방향을 제안한 프로젝트입니다. 트위터 데이터를 크롤링해 어간 추출·품사 필터링(명사/형용사) 기반 키워드 빈도를 산출했고, 워드클라우드로 고객 수요(영상/음원 기록물 니즈 등)를 시각화했습니다.
