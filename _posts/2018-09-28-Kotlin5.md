## 조건문
----

#### 1. if

Kotlin에서 if문은 일반적인 사용과 표현식으로서의 사용이 가능합니다.  

```
// 일반적인 사용
fun main(args: Array<String>) {
	val num = 7

	if (num < 10) println("$num < 10")

	if (num > 0 && num <= 10) {
		println("0 < $num <= 10")
	} else if (num > 10 && num <= 20) {
		println("10 < $num <= 20")
	} else {
		println("20 < $num")
	}
}

>>> 출력결과
7 < 10
0 < 7 <= 10
```

<br>

if문을 표현식으로 사용하게 되면 특정한 값이 발생하게 됩니다.  
따라서 다음과 같은 작업이 가능해집니다.
- 대입가능
- 함수의 전달인자로 사용 가능

```
// 표현식으로 사용
fun main(args: Array<String>) {
	val num = 7

	val ans = if(num > 10) "yes" else "no"

	println("$num > 10 ? " + ans)
}

>>> 출력결과
7 > 10 ? no
```

<br>

if문을 표현식으로 사용할 때 몇 가지 주의사항이 있습니다.  

- 중괄호`{}`를 이용해 여러 줄 작성하는 경우 if 표현식으로 발생하는 데이터는 **맨 마지막줄**에 작성합니다.
  ```
  fun main(args: Array<String>) {
  	val n = 7
  	val re = if(n < 10){
  		println("$n < 10")
  		10 - n
  	} else {
  		println("$n > 10")
  		n - 10
  	}

  	println("차이: $re")
  }

  >>> 출력결과
  7 < 10
  차이: 3
  ```
- else가 없으면 에러가 발생합니다.

#### 2. when

Kotlin의 when은 switch-case문의 확장판으로 생각하면 편합니다.  
역할은 비슷하지만 보다 기능이 많습니다.  

- 정수 값으로 분기

  - switch-case문과 같이 정수로 분기할 수 있습니다.
  - 마지막의 else는 생략할 수 있습니다.

```
val n = n

when (n) {
	1 -> println("n == 1")
	2 -> println("n == 2")
	3 -> println("n == 3")
  else -> println("n > 3")
}
```

<br>

- 정수 외의 값으로 분기

  - 정수 외에 다른 타입으로도 분기할 수 있습니다.

```
val str = "Morning"

when (str) {
	"Morning" -> println("아침")
	"Lunch" -> println("점심")
	"Diner" -> println("저녁")
}
```

<br>

- 여러 값으로 분기

  - `,`를 이용해 여러 값을 하나의 범주로 묶어 분기할 수 있습니다.

```
val str = "y"

when (str) {
	"yes", "y", "o" -> println("yes")
	"no", "n", "x" -> println("no")
}
```

<br>

- 범위 값으로 분기

  - 예약어 `in`을 이용하면 범위 내의 값으로 분기할 수 있습니다.
  - `!in`을 이용하면 범위 외의 값으로 분기할 수 있습니다.
  - 컬렉션 타입(list, array 등)으로 범위를 지정할 수 있습니다.

```
val year = 17

when (year) {
	!in 8..19 -> println("의무교육 대상 아님")
	in 8..13 -> println("초등학생")
	in 14..16 -> println("중학생")
	in 17..19 -> println("고등학생")
}
```

<br>

- 여러 타입으로 분기

  - Any 타입으로 데이터를 받으면 여러 타입으로 분기할 수 있습니다.
  - 예약어 `is`를 사용하면 자료형으로 분기할 수 있습니다.

```
fun whenTest(data: Any){
	when(data){
		1 -> println("Int")
		"Hello" -> println("String")
		is Boolean -> println("Boolean")
	}
}
```

- when 역시 if문처럼 **표현식**으로 사용할 수 있습니다.

<br>

## 반복문
----
#### 1. for

Kotlin에서 for문의 사용법은 Python과 닮아있습니다.  
기존의 C, Java등과는 차이가 있습니다.

- 범위 내에서 반복할 수 있습니다.
```
fun main(args: Array<String>){
	var sum = 0

	for(i in 1..10){
    // 1 ~ 10 사이에서 1씩 증가하며 i에 대입됩니다.
		sum += i
	}
	println("sum: $sum")
}
```
`>>> sum: 55`

<br>

- 범위를 설정하는 다양한 방법이 있습니다.
```
for(i in 1..10)         // 1 ~ 10 사이
for(i in 1 until 10)    // 1 ~ 10 미만
for(i in 2..10 step 2)  // 2 ~ 10 사이 2씩 증가
for(i in 10 downTo 1)   // 10 ~ 1 사이 감소
```

<br>

- 컬렉션 타입(array, list 등)에 포함된 데이터를 하나씩 추출하여 for문을 실행할 수 있습니다.
```
fun main(args: Array<String>){
	var list = listOf("Hi", "Hello", "Yo")

	for(str in list){
		print(str + " ")
	}
}
```
`>>> Hi Hello Yo `

<br>

- 프로퍼티 `indices`으로 컬렉션 타입의 인덱스값을 이용할 수 있습니다.
```
fun main(args: Array<String>){
	var list = listOf("Hi", "Hello", "Yo")

	for(i in list.indices){
		println("list[$i]: " + list[i])
	}
}
```
```
>>> list[0]: Hi
>>> list[1]: Hello
>>> list[2]: Yo
```

<br>

- `withIndex()`함수를 사용하면 컬렉션 타입의 데이터와 인덱스값 모두 이용할 수 있습니다.
```
fun main(args: Array<String>){
	var list = listOf("Hi", "Hello", "Yo")

	for((idx, value) in list.withIndex()){
		println("list[$idx]: $value")
	}
}
```
```
>>> list[0]: Hi
>>> list[1]: Hello
>>> list[2]: Yo
```

<br>

#### 2. while

while문의 경우 일반적인 쓰임과 같습니다.  
```
fun main(args: Array<String>){
	var i = 0

	while(i < 3) {
		print(i)
		i++
	}
}
```
`012`

break와 continue 역시 마찬가지입니다.  
- break는 블록`{}`을 벗어납니다.
- continue는 블록`{}`의 맨 마지막 줄로 이동합니다.
```
fun main(args: Array<String>){
	var i = 0

	while(true) {
		print(i)
		i++
		if (i > 5) break
	}
}
```
`>>> 012345`

반복문을 중첩해서 사용했을 경우, 기본적으로 해당 블록에 영향을 줍니다.  
라벨`@`을 이용하면 바깥의 블록을 벗어날 수 있습니다.
- 벗어날 구문의 앞에 `labelName@`를 입력하고, break나 continue 뒤에 `@labelName`을 입력해 사용합니다.
```
fun main(args: Array<String>){
	var i = 0

	aaa@ while(true) {
		for(j in 1..10) {
			println("inner $j")
			if(j >= 5) break@aaa
		}

		println("outter $i")
	}
}
```
```
>>> inner 1
>>> inner 2
>>> inner 3
>>> inner 4
>>> inner 5
```
